# حزمة إدارة التراخيص البرمجية المستقلة (AQNEX Licensing System)

هذه الحزمة عبارة عن نظام مستقل لإصدار التراخيص الرقمية والتحقق منها، وهي جاهزة لإعادة الاستخدام والدمج المباشر في أي نظام برمجيات PHP آخر تبيعه لعملائك ليعمل بنفس الآلية التلقائية والأكواد النصية.

---

## 1. محتويات المجلد:
1. **`Licensing.php`**: الكود البرمجي الأساسي للتحقق وتوليد معرف الجهاز (Machine ID) المعتمد على عتاد الجهاز (ويندوز).
2. **`activate.php`**: شاشة التنشيط للعميل (مستقلة تماماً وتعتمد على سحابة Bootstrap و jQuery CDNs). تتيح للعميل استخراج ملف الطلب ولصق كود التفعيل والتحقق الفوري بالـ AJAX.
3. **`license_generator.php`**: لوحة التحكم الخاصة بالشركة (المسؤول) لتوليد مفاتيح تشفير RSA جديدة، استيراد وقراءة طلب العميل، وإصدار أكواد التفعيل النصية.

---

## 2. خطوات دمج الحزمة في أنظمة برمجية أخرى:

### الخطوة 1: نسخ المجلد
قم بنسخ مجلد `licensing_system` بأكمله ووضعه داخل مجلد مشروعك الجديد.

### الخطوة 2: توليد مفاتيح تشفير جديدة للمشروع الجديد
1. افتح لوحة التحكم الخاصة بك في المتصفح: `http://localhost/YOUR_PROJECT/licensing_system/license_generator.php`.
2. انزل إلى أسفل الصفحة واضغط على **"توليد زوج مفاتيح RSA جديد كلياً"** (سيقوم المولد بحفظ ملف المفتاح الخاص `private.key` سرياً في مجلدك، وسيظهر لك **المفتاح العام** في الصندوق النصي).

### الخطوة 3: تضمين المفتاح العام في كلاس التحقق
1. انسخ المفتاح العام الظاهر بالكامل.
2. افتح ملف `Licensing.php` الجديد.
3. قم بلصقه كقيمة للمتغير `$publicKey` (بدون أي مسافات بادئة في بداية السطور لتجنب أخطاء openssl).

### الخطوة 4: حماية صفحات نظامك الأساسي
في بداية ملف التوجيه أو ملف الإعداد العام للمشروع الجديد (على سبيل المثال `bootstrap.php` أو `header.php` أو `index.php`)، ضع الكود التالي لحظر الدخول إلا بوجود ترخيص صالح:

```php
<?php
// تضمين ملف التراخيص
require_once(__DIR__ . '/licensing_system/Licensing.php');

$licensing = new \AQNEX\Licensing\Licensing(__DIR__ . '/licensing_system/license.AQNEX');

$verify = $licensing->verifyLicense();
if (!$verify['status']) {
    // إذا كان العميل على صفحة التفعيل، دعه يكمل التنشيط
    $current_page = basename($_SERVER['PHP_SELF']);
    if ($current_page !== 'activate.php') {
        // التوجيه الفوري لصفحة التفعيل
        header("Location: licensing_system/activate.php");
        exit();
    }
}
?>
```

### الخطوة 5: حفظ التفعيل في قاعدة بيانات العميل (اختياري)
داخل ملف `licensing_system/activate.php` ستجد تعليق **TODO** واضح ومحدد، يمكنك إلغاء تعليقه لتوصيل قاعدة بيانات النظام الجديد وتسجيل بيانات الرخصة محلياً (مثل نوع الباقة وتاريخ الانتهاء والموديولات المفعلة) لخدمة الصلاحيات الداخلية في مشروعك الجديد.
